A few of my projects.

Here are some projects I've worked on as a designer and developer.

Explore Eastern Utah.

I built Explore Eastern Utah to showcase local restaurants that remained open during pandemic shutdowns. I designed it in Adobe XD and made it using WordPress and Bootstrap. There is a lot of dynamic content to allow business owners to update their pages. I designed all logos and graphics in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Arcane Ops. Paranormal Podcast.

Arcane Ops was a limited-run paranormal podcast that ran for seven episodes at the end of 2020. I designed it in Adobe XD and built it using WordPress, WooCommerce, and Bootstrap. I designed all logos and graphics in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Corporate Law Website.

Simple brochure website I designed and built for a corporate law firm. I created the UI in Adobe XD and developed it using Webflow, Finsweet Client First, and custom JS components. I like Webflow's interface and ease of use, but it's a bit restrictive for complex styling. I designed all logos and graphics in Adobe CC.

Webflow, HTML, CSS, JavaScript, Adobe CC

Sheriff Campaign Website.

Campaign website for a sheriff election. This website had many pages, a simple CMS for news updates, and simple PayPal API setup to accept donations. I created the UI in Adobe XD and built it using Bootstrap, HTML, CSS, JavaScript, and some basic PHP to run the CMS. I designed all logos and graphics in Adobe CC.

HTML, CSS, JavaScript, PHP, Bootstrap, Adobe CC
