A few of my projects.

Here are some projects I've worked on as a designer and developer.

Unredacted Magazine.

I designed the entire layout of Unredacted Magazine, beginning with Issue 002. I built an adaptable, accessible layout in InDesign. In addition, I created several graphical components and ads in Illustrator. Since the magazine is digital-only, I exported the final version as an interactive PDF.

Adobe InDesign, Illustrator

Fortify 24x7 Magazine Ads.

I designed two half-page magazine ads for Fortify 24x7. After receiving the parameters and copy, I designed the ads in Adobe Illustrator. Since the magazine is an Interactive PDF, I placed the final designs in an InDesign document and embedded multiple targeted hyperlinks. These ads can be seen in Issue 002 of Unredacted Magazine.

Adobe Illustrator, InDesign

Explore Eastern Utah.

I built Explore Eastern Utah to showcase local restaurants that remained open during pandemic shutdowns. I designed the UI in Adobe XD and built it using WordPress and Bootstrap. There is a lot of dynamic content to allow business owners to update their pages. All logos and graphics were designed in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Arcane Ops. Paranormal Podcast.

Arcane Ops was a limited-run paranormal podcast that ran for seven episodes at the end of 2020. I designed the layout in Adobe XD and built it using WordPress, WooCommerce, and Bootstrap. All logos and graphics were designed in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Corporate Law Website.

Simple brochure website I designed and built for a corporate law firm. I created the UI in Adobe XD and developed it using Webflow, Finsweet Client First, and custom JS components. Webflow's interface and ease of use are intuitive, but it's a bit restrictive for complex styling. All logos and graphics were designed in Adobe CC.

Webflow, HTML, CSS, JavaScript, Adobe CC

Sheriff Campaign Website.

Campaign website for a sheriff election. This website had many pages, a simple CMS for news updates, and simple PayPal API setup to accept donations. I created the UI in Adobe XD and built it using Bootstrap, HTML, CSS, JavaScript, and some basic PHP to run the CMS. All logos and graphics were designed in Adobe CC.

HTML, CSS, JavaScript, PHP, Bootstrap, Adobe CC
